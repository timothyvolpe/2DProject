Default Lua Library Notes

Base

  • print - Overridden by game, behavior should be almost identical.
  • error - Overridden by game, behavior should be almost identical.

io

Disabled for security reasons, functionality will eventually be partially restored.

os

Disabled for security reasons, functionality may eventually be partially restored.

Useful Global Variables

  • CLIENT - True when the script is running on the client, false otherwise.
  • SERVER - True when the script is running on the server, false otherwise.

Global Functions

  • include( path ) - includes a file's contents
    • Arguments: path, the path of the file to include relative to the current file
    • Returns: ignored
    • Version added: 0.2
    • Version removed: N/A

Game Class

User definable functions

  • Initialize( ) - the entry point, called when the game runs the lua file. For the client, this is when the game start; for the server, this is when the server starts the lua loading stage.
    • Arguments: none
    • Returns: ignored
    • Version added: 0.2
    • Version removed: N/A

Client-sided functions

  • GetResolution( ) - returns the games current resolution
    • Arguments: none
    • Returns: x, y - Resolution of the game window in pixels
    • Version added: 0.2
    • Version removed: N/A
  • GetResolutionX( ) - returns the games current resolution in the X dimension
    • Arguments: none
    • Returns: x - X resolution of the game window in pixels
    • Version added: 0.2
    • Version removed: N/A
  • GetResolutionY( ) - returns the games current resolution in the Y dimension
    • Arguments: none
    • Returns: y - Y resolution of the game window in pixels
    • Version added: 0.2
    • Version removed: N/A
  • IsFullscreen( ) - returns a boolean describing the fullscreen state of the game
    • Arguments: none
    • Returns: fullscreen - true if fullscreen, false otherwise
    • Version added: 0.2
    • Version removed: N/A
  • IsBorderless( ) - returns a boolean describing the borderless state of the game. Meaningless when in fullscreen
    • Arguments: none
    • Returns: borderless - true if borderless, false otherwise. Unknown if fullscreen.
    • Version added: 0.2
    • Version removed: N/A

Interface Class

Client-sided functions

  • Interface.new( type, name, [parent] ) - static, the interface class constructor
    • Arguments: type, the type of interface control to create, name, the unique name of the interface control, and optionally parent which is the controls parent Interface object
    • Returns: userdata - a userdata type pointing to the class object
    • Version added: 0.2
    • Version removed: N/A

    • Valid interface controls are "Label" and "Panel". Case sensitive.

  • Interface.GetControl( name ) - static, gets a control by its name
    • Arguments: name, the name of the control to get
    • Returns: userdata - a userdata type pointing to the control, or nil if it was not found
    • Version added: 0.2
    • Version removed: N/A
  • activate() - activates the interface object, must be called for it to be usable
    • Arguments: none
    • Returns: success - true if activation false successful, false otherwise
    • Version added: 0.2
    • Version removed: N/A
